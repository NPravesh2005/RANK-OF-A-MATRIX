## Algorithm:

### Step 4: Run the program

## Program: 

```python

#Program to find the rank of a matrix.
#Developed by: N.PRAVESH
#RegisterNumber: 212223230154

import numpy as py
a = py.array([[1,2,3],[3,6,9]])
rank = py.linalg.matrix_rank(a)
print(rank)

```
